- More organize option
- Github Flavored Markdown(with Latex)
- Using PouchDB(competible CouchDB and  any LevelDB adapter)

- Electron
- React
- React Router
- Styled Components
- Redux
- Webpack
- PouchDB / LevelDB
- wooorm/Remark
- Katex

- Octicon
- Github Markdown CSS
